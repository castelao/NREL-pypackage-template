# Python Package Template

## Quick start

This repository is a template for creating a Python package, providing some common basic configurations and tools. To use it,

1. Go to: https://github.nrel.gov/Data-Engineering-SEAC/python-package
2. Click on the green "Use this template" button.
3. Select Create a new repository:
<img src="misc/use-this-template-button.webp" alt="Create a new repository" width="400"/>
4. Follow the same steps as creating a regular repository, such as defining owner, repository name, ...

## Deep dive into the template

For more details on the tools used and options made in this template, please
check the [wiki](https://github.nrel.gov/Data-Engineering-SEAC/python-package/wiki).
